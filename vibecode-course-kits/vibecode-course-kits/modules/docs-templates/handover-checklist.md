# Handover Checklist

> Fill before Demo Day / client handoff.

## Live product

- [ ] Public URL: `________________`
- [ ] Works on mobile browser
- [ ] Form / data flow tested with real sample input

## Documentation

- [ ] `README.md` describes what the product is and how to use it
- [ ] Env / keys instructions (if any) — **never commit secrets**
- [ ] Contact / support note for the owner

## Security basics

- [ ] No API keys in public frontend code
- [ ] Database rules reviewed (read/write permissions)

## Demo script (3–5 minutes)

1. Show landing page
2. Trigger main interaction / form
3. Show data result (dashboard or console / cloud)
4. Share live link
